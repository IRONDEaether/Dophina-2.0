# Dophina-2.0
Un shérif du code et démon protecteur en Python qui audite la sécurité réelle des sites web. 
A code sheriff and protective demon in Python that audits the real security of websites

Dophina 2.0 is a network and web security auditing tool built with Python. Unlike traditional scanners that provide a false sense of security with biased marketing scores, Dophina enforces the strict justice of code.

### ⚖️ The 24% Logic (The Hacker Trap)
In Dophina 2.0, the maximum security score for any active and functional website is intentionally capped at **24%**.
* **Why?** Because 0% risk does not exist in the real world. A website reaching 24% is a true streamlined fortress (zero dangerous JavaScript, ironclad network headers, ultra-secure forms).
* **The Strategy:** A hacker scanning the site will assume a 24% score means the target is highly vulnerable. Upon trying to attack, they will slam into an invisible wall. It creates a total psychological counter-play to trap attackers.

### 🛠️ Project Architecture
The project is split into two distinct scripts to maximize performance and code cleanliness:
* `interface.py`: The graphical user interface (GUI), compiled with the console hidden.
* `S_2.PY`: The backend network scanning engine and logical server running in the background.
